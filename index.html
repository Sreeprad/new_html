GIT Workshop

Welcome to the GIT Workshop! This workshop aims to introduce you to the fundamentals of Git, a distributed version control system. Git is widely used in software development to manage source code and collaborate with teams effectively.

During this workshop, you will learn the following key concepts:

  • Creating a new Git repository
  • Tracking changes using Git
  • Branching and merging
  • Collaborating with others using Git
  • Resolving conflicts

Whether you are a beginner or have some experience with Git, this workshop will provide you with valuable insights and practical hands-on exercises to enhance your Git skills. We encourage you to bring your laptop with Git installed so that you can actively participate in the workshop activities.

Join us for an informative and interactive Git Workshop that will equip you with the knowledge and tools to become proficient in using Git for your projects.
